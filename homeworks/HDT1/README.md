# Hoja de Trabajo 1 – Agentes Inteligentes y Entornos

## Objetivo
Aplicar los conceptos fundamentales de agentes inteligentes para analizar escenarios
reales de diseño de sistemas, enfocándose en racionalidad, métricas de desempeño,
modelado y arquitectura de agentes.

## Contenido
- Análisis de racionalidad y métricas de desempeño
- Modelado PEAS y clasificación de entornos
- Representación de estados, inferencia y aprendizaje
- Implementación básica de un agente reflejo simple

## Tecnologías
- Python (programación orientada a objetos)
