# Laboratorios

Esta carpeta contiene los **laboratorios prácticos** del curso de Inteligencia Artificial.

Cada laboratorio se enfoca en la implementación y experimentación de un concepto específico
visto en clase, incluyendo aprendizaje automático, búsqueda, planificación y razonamiento
bajo incertidumbre.

Los laboratorios están organizados por tema y numerados de forma secuencial.
