# Laboratorio 2 – Clasificación y Limitaciones de Modelos

## Objetivo
Comprender las limitaciones matemáticas de los modelos de clasificación y construir
clasificadores desde cero para entender cómo aprende realmente una máquina.

## Contenido
- Convexidad y función de costo en regresión logística
- Problemas de desbalance de clases en KNN
- Overfitting y bias-variance tradeoff
- Implementación manual de:
  - Regresión logística con descenso del gradiente
  - K-Nearest Neighbors
- Comparación con modelos de scikit-learn

## Tecnologías
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn (solo para benchmarking)
