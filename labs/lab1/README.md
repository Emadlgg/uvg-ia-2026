# Laboratorio 1 – Métricas, Datos y Evaluación de Modelos

## Objetivo
Analizar decisiones de diseño en sistemas de aprendizaje automático y comprender
las consecuencias prácticas de métricas, balance de datos y evaluación de modelos.

## Contenido
- Análisis de métricas (Precision, Recall, Accuracy, R² Ajustado)
- Simulación y limpieza manual de datasets
- Manejo de datos faltantes y clases desbalanceadas
- Implementación manual de métricas de error (MAE y RMSE)

## Tecnologías
- Python
- NumPy
- Pandas
